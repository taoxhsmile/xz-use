# useHover

一个用于追踪 dom 元素是否有鼠标悬停的 Hook。

Api 与ahooks一致。

# 基本用法

# 传入 Dom 元素

# API

const isHovering = useHover(target, {
  onEnter,
  onLeave,
});

# 结果

参数说明类型
isHovering判断鼠标元素是否处于 hover 元素boolean

# 参数

参数说明类型默认值
targetDOM 节点或者 Ref 对象(() => HTMLElement) | HTMLElement | React.RefObject-
onEnter监听进入 hover()=>void-
onLeave监听离开 hover()=>void-
